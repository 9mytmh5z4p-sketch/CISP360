//Casting Workshop

You are creating a multi menu program to activate static casts in C++. You set the types and collect the data with the two sub menus, but you need to keep track of the state so that you can have the cast activate the appropriate types. Only 1 menu collects the data to cast.

Create a main menu with the following options:
1. To type
2. From type
3. Cast!
4. Quit

Options 1 & 2 trigger sub menus:
Option 1 & 2 Submenus - each a different menu, but look the same.
1. Int
2. Float
3. Short
