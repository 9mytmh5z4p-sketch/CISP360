Displaying Output with cout

Target duration: ~15 minutes - Basic concepts only

Agenda

Today’s topics:

  • What is cout and why do we need it?
  • Basic syntax and the << operator
  • Displaying text, numbers, and variables
  • Chaining output with <<
  • Creating new lines with endl and \n
  • Common mistakes to avoid

This is the foundational lecture. Advanced formatting, escape sequences, and patterns will be covered in a follow-up lecture.

What is cout?

cout (pronounced “see-out”) stands for “console output”

Purpose: Sends data to the screen (the console/terminal)

Requirements: - Part of the <iostream> library - In the std namespace

Think of cout as your program’s voice - it’s how your program “talks” to the user.

Emphasize that without cout, programs can’t communicate results to users. Show the connection between #include and being able to use cout.

Basic Syntax

The basic syntax uses the stream insertion operator <<

cout << "something to display";

Read it as: “send to cout” or “insert into cout”

The << operator is like an arrow pointing toward cout, showing that data flows TO the output stream.

Draw an arrow showing data flowing from right to left into cout. Emphasize that << is an operator, not just syntax decoration.

Your First Program

#include <iostream>
using namespace std;

int main()
{
    cout << "Hello, World!";
    return 0;
}

Output:

Hello, World!

What’s happening: - #include <iostream> brings in the iostream library (where cout lives) - using namespace std; lets us write cout instead of std::cout - cout << "Hello, World!"; sends the text to the screen

Displaying Text

Text must be enclosed in double quotes " "

cout << "Welcome to C++ Programming!";

Output:

Welcome to C++ Programming!

Rule: Text (string literals) always needs double quotes.

Common mistake: Students forget the quotes. Show what happens when you omit them - compiler thinks it’s a variable name.

Displaying Numbers and Variables

Numbers don’t need quotes:

cout << 42;
cout << 3.14159;

Variables don’t need quotes either:

int age = 25;
string name = "Alice";

cout << age;    // Displays: 25
cout << name;   // Displays: Alice

Key distinction: Quotes = literal text, No quotes = variable or number

Chaining Output

You can display multiple things in one statement by chaining with <<

cout << "Hello" << " " << "World";

Output:

Hello World

Mixing text and variables:

string name = "Bob";
int age = 30;

cout << "My name is " << name << " and I am " << age << " years old.";

Output:

My name is Bob and I am 30 years old.

Emphasize that you need to include spaces in your quoted strings. Show what happens without the space in ” and I am “.

Adding New Lines

By default, cout doesn’t move to the next line. Use endl or \n:

Method 1: Using endl

cout << "First line" << endl;
cout << "Second line" << endl;
cout << "Third line" << endl;

Method 2: Using \n

cout << "First line\n";
cout << "Second line\n";
cout << "Third line\n";

Both produce:

First line
Second line
Third line

endl vs \n

endl - Moves to next line AND flushes the output buffer

\n - Just moves to next line (slightly faster)

For now: Use whichever you prefer. Both work the same for simple programs.

Creating blank lines:

cout << "Paragraph 1" << endl;
cout << endl;              // Blank line
cout << "Paragraph 2" << endl;

Output:

Paragraph 1

Paragraph 2

Common Mistakes

Mistake 1: Forgetting quotes around text

cout << Hello;   // ERROR - thinks Hello is a variable
cout << "Hello"; // CORRECT

Mistake 2: Using single quotes for text

cout << 'Hello';  // ERROR - single quotes for ONE character only
cout << "Hello";  // CORRECT - double quotes for text

Mistake 3: Confusing variables and text

int age = 25;
cout << "age";  // Displays: age (the word)
cout << age;    // Displays: 25 (the value)

Getting User Input with cin

Introduction

So far, we’ve been using cout to output information to the screen. But what if we want our programs to be interactive?

That’s where cin comes in!

Think of it this way:

  • cout = output (computer talks to user) → screen
  • cin = input (user talks to computer) ← keyboard

What is cin?

cin (pronounced “see-in”) stands for “character input” or “console input”

  • It reads data from the keyboard
  • It’s part of the <iostream> library (same as cout)
  • It’s in the std namespace

Syntax:

cin >> variableName;

Think of the arrows:

  • cout << - arrows point toward cout (data flows TO output)
  • cin >> - arrows point away from cin (data flows FROM input)

Simple Example

Let’s write a program that asks for the user’s age:

#include <iostream>
using namespace std;

int main()
{
    int age;  // Declare a variable to store the age
    
    cout << "How old are you? ";  // Ask the question
    cin >> age;                    // Wait for user input
    
    cout << "You are " << age << " years old." << endl;
    
    return 0;
}

What happens: Program displays the prompt, waits for input, stores the value in age, then displays the result.

Key Rules for Using cin

1. Declare the Variable First

WRONG:

cin >> age;  // Error! What is 'age'?

RIGHT:

int age;     // Declare the variable first
cin >> age;  // Now we can read into it

2. Variable Type Matters

int number;      // Expects whole numbers
cin >> number;   

double price;    // Expects decimal numbers
cin >> price;    

string name;     // Expects text
cin >> name;

Reading Different Data Types

// Reading an integer
int numberOfPets;
cout << "How many pets do you have? ";
cin >> numberOfPets;

// Reading a decimal
double temperature;
cout << "What is the temperature? ";
cin >> temperature;

// Reading a single word
string firstName;
cout << "What is your first name? ";
cin >> firstName;

Important: cin >> only reads one word at a time. It stops at whitespace (spaces, tabs, Enter).

Reading Multiple Values

You can read multiple values in one statement:

int month, day, year;

cout << "Enter your birth month, day, and year: ";
cin >> month >> day >> year;

cout << "You were born on " << month << "/" << day << "/" << year << endl;

User types: 3 15 2005 (separated by spaces) and presses Enter

Output: You were born on 3/15/2005

Complete Example Program

// input_demo.cpp - Demonstrating cin
#include <iostream>
using namespace std;

int main()
{
    string name;
    int age;
    double height;
    
    cout << "Enter your name: ";
    cin >> name;
    
    cout << "Enter your age: ";
    cin >> age;
    
    cout << "Enter your height in inches: ";
    cin >> height;
    
    cout << "\nName: " << name << endl;
    cout << "Age: " << age << " years old" << endl;
    cout << "Height: " << height << " inches" << endl;
    
    return 0;
}

Common Mistakes

Mistake 1: Forgetting to Declare

cin >> age;  // Error! 'age' doesn't exist

Mistake 2: No Prompt Message

cin >> age;  // User doesn't know what to enter!

Mistake 3: Using >> for Multi-word Input

string fullName;
cin >> fullName;  // User types "John Smith"
// Only captures "John" - stops at space!

We’ll learn about getline() for multi-word input in an advanced lecture.

Good Practices

Always Prompt Before Input

cout << "Enter the temperature in Fahrenheit: ";
cin >> value;

Echo the Input Back

int quantity;
cout << "How many items? ";
cin >> quantity;
cout << "You entered: " << quantity << endl;

Use Meaningful Variable Names

int numberOfStudents;  // Good
cin >> numberOfStudents;

int x;  // Bad - unclear
cin >> x;

What Are Manipulators?

Manipulators are special functions that modify how cout displays data.

Purpose: Control the formatting of output (not what to display, but how to display it)

Requirements: - Basic manipulators: #include <iostream> (endl, boolalpha, fixed) - Advanced manipulators: #include <iomanip> (setprecision, setw, setfill)

#include <iostream>
#include <iomanip>  // Always include for formatting!
using namespace std;

Emphasize that manipulators are instructions that change how cout behaves. They’re not data themselves.

Decimal Precision: setprecision(n)

Controls how many digits are displayed.

Without fixed: Controls total significant digits

double pi = 3.14159265;
cout << setprecision(3) << pi << endl;  // 3.14
cout << setprecision(5) << pi << endl;  // 3.1416

With fixed: Controls decimal places (digits after decimal)

double price = 19.5;
cout << fixed << setprecision(2) << price << endl;  // 19.50

Rule: Always use fixed with setprecision for predictable formatting.

The fixed Manipulator

fixed forces fixed-point notation (regular decimals, not scientific notation).

Without fixed:

double num = 123.456789;
cout << setprecision(2) << num << endl;  // 1.2e+02 (scientific!)

With fixed:

double num = 123.456789;
cout << fixed << setprecision(2) << num << endl;  // 123.46

Best practice: Use fixed whenever formatting decimal numbers for users.

Show that without fixed, large or small numbers may appear in scientific notation, which confuses users.

Money Formatting Pattern

Always use fixed with setprecision(2) for money:

double price = 29.5;
double tax = 2.3625;
double total = price + tax;

cout << fixed << setprecision(2);  // Set once for all output

cout << "Price: $" << price << endl;    // Price: $29.50
cout << "Tax:   $" << tax << endl;      // Tax:   $2.36
cout << "Total: $" << total << endl;    // Total: $31.86

Note: fixed and setprecision persist until changed.

Boolean Formatting: boolalpha

By default, booleans display as numbers (1 or 0). Use boolalpha for words.

Without boolalpha:

bool isStudent = true;
bool hasLicense = false;

cout << "Student: " << isStudent << endl;   // Student: 1
cout << "License: " << hasLicense << endl;  // License: 0

With boolalpha:

cout << boolalpha;
cout << "Student: " << isStudent << endl;   // Student: true
cout << "License: " << hasLicense << endl;  // License: false

Tip: Use noboolalpha to switch back to numbers.

Field Width: setw(n)

setw(n) sets the minimum width for the next output item only.

int num = 42;

cout << setw(5) << num << endl;    //    42 (5 chars wide, right-aligned)
cout << setw(10) << num << endl;   //        42 (10 chars wide)
cout << num << endl;               // 42 (setw reset!)

Critical: setw() only affects the NEXT item, then resets automatically.

Emphasize this is the most common mistake - students forget setw resets and wonder why their columns don’t align.

Creating Aligned Columns

Use setw() for each column item:

cout << setw(10) << "Name" << setw(8) << "Age" << setw(10) << "Grade" << endl;
cout << setw(10) << "Alice" << setw(8) << 20 << setw(10) << 95 << endl;
cout << setw(10) << "Bob" << setw(8) << 22 << setw(10) << 88 << endl;

Output:

      Name     Age     Grade
     Alice      20        95
       Bob      22        88

Show a diagram of how setw creates invisible boxes for each item. Point out that by default, content is right-aligned.

Alignment: left and right

By default, output is right-aligned. Use left to change this.

Right-aligned (default):

cout << setw(10) << "Hello" << endl;  //      Hello

Left-aligned:

cout << left;
cout << setw(10) << "Hello" << endl;  // Hello

Switch back:

cout << right;  // Return to default

Note: left and right persist until changed.

Practical Table Example

cout << fixed << setprecision(2);
cout << left;

cout << setw(15) << "Item" << right << setw(10) << "Price" << endl;
cout << left << setw(15) << "Coffee" << right << setw(10) << "$3.50" << endl;
cout << left << setw(15) << "Sandwich" << right << setw(10) << "$7.99" << endl;

Output:

Item                 Price
Coffee               $3.50
Sandwich             $7.99

Walk through how we switch between left and right alignment for different columns.

Fill Characters: setfill(c)

setfill(c) sets the character used to fill empty space in fields.

Default is space:

cout << setw(10) << 42 << endl;  //         42

Custom fill:

cout << setfill('0') << setw(5) << 42 << endl;     // 00042
cout << setfill('*') << setw(10) << "Hi" << endl;  // ********Hi

Note: setfill persists until changed.

Practical Fill Character Uses

Invoice numbers with leading zeros:

int invoiceNum = 237;
cout << setfill('0') << setw(6) << invoiceNum << endl;
// Output: 000237

Decorative separator lines:

cout << setfill('-') << setw(40) << "" << endl;
// Output: ----------------------------------------

Remember: Reset fill to space when done: setfill(' ')

Persistent vs. Temporary Manipulators

Persistent (stay active until changed): - fixed, scientific - setprecision(n) - boolalpha, noboolalpha - left, right - setfill(c)

Temporary (reset after one use): - setw(n) - Only affects next output!

Create a visual showing persistent manipulators as switches that stay on, versus setw as a one-time-use token.

Other Useful Manipulators

showpoint / noshowpoint: Control trailing zeros

cout << showpoint << 5.0 << endl;  // 5.00000

scientific: Scientific notation

cout << scientific << 123456789.0 << endl;  // 1.234568e+08

uppercase / nouppercase: Affects scientific notation

cout << scientific << uppercase << 1234.5 << endl;  // 1.2345E+03

Complete Example: Receipt Formatter

#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    double coffee = 3.50, muffin = 2.75, sandwich = 7.99;
    double subtotal = coffee + muffin + sandwich;
    double tax = subtotal * 0.08;
    double total = subtotal + tax;
    
    cout << fixed << setprecision(2) << left;
    
    cout << "====== RECEIPT ======" << endl;
    cout << setw(15) << "Coffee" << "$" << right << setw(6) << coffee << endl;
    cout << left << setw(15) << "Muffin" << "$" << right << setw(6) << muffin << endl;
    cout << left << setw(15) << "Sandwich" << "$" << right << setw(6) << sandwich << endl;
    cout << setfill('-') << setw(23) << "" << setfill(' ') << endl;
    cout << left << setw(15) << "Subtotal" << "$" << right << setw(6) << subtotal << endl;
    cout << left << setw(15) << "Tax (8%)" << "$" << right << setw(6) << tax << endl;
    cout << setfill('=') << setw(23) << "" << setfill(' ') << endl;
    cout << left << setw(15) << "TOTAL" << "$" << right << setw(6) << total << endl;
    
    return 0;
}

Common Mistakes to Avoid

Mistake 1: Forgetting #include <iomanip>

cout << setprecision(2) << 3.14;  // ERROR without <iomanip>

Mistake 2: Using setprecision() without fixed

cout << setprecision(2) << 19.5;  // Unpredictable!

Common Formatting Patterns

Money formatting:

cout << fixed << setprecision(2) << "$" << amount;

Percentage formatting:

cout << fixed << setprecision(2) << (rate * 100) << "%";

Table headers:

cout << left << setw(15) << "Header1" << setw(10) << "Header2" << endl;

Boolean status:

cout << boolalpha << "Active: " << isActive << endl;

Zero-padded numbers:

cout << setfill('0') << setw(6) << number << setfill(' ');

Creating Formatted Tables in C++

~ 30 slides

Introduction

You’ve learned how to format individual values with manipulators. Now let’s combine those skills to create something professional: formatted tables!

Tables are everywhere in programming:

  • Financial reports
  • Data summaries
  • System information displays
  • Database outputs

Key Question: How do we decide which table format to use?

Answer: The data inside the table drives the decision about which table to use.

Why Tables Matter

No formatting

Alice 30 50000.5
Bob 25 45000.8
Charlie 28 52000

With table formatting

=====================================
| Name      | Age  | Salary        |
=====================================
| Alice     | 30   | 50000.5       |
| Bob       | 25   | 45000.8       |
| Charlie   | 28   | 52000         |
=====================================

Which is easier to read?

The Four Essential Table Types

We’ll cover four common table patterns, each suited for different types of data:

  1. Fixed Width Table with Borders - Professional reports with clear separation
  2. Hexadecimal and Octal Table - Number system conversions
  3. Cash Table with Negative Numbers - Financial data with parentheses for negatives
  4. Scientific Notation Table - Very large or very small numbers

Table Type 1: Fixed Width Table with Borders

When to Use

  • Clear visual separation between columns
  • Professional appearance
  • Mixed data types (strings and numbers)
  • General-purpose data display

Example Output

Fixed Width Table with Borders
================================================
| Name         | Age        | Salary           |
================================================
| Alice        | 30         | 50000.5          |
| Bob          | 25         | 45000.8          |
| Charlie      | 28         | 52000            |
================================================
End of Report.

Fixed Width Table: Key Techniques

  1. Define column widths as constants
const int nameWidth = 15;
const int ageWidth = 10;
const int salaryWidth = 12;
  1. Use setfill() to create border lines
cout << setfill('=') << setw(totalWidth) << "=" << endl;
  1. Use setw() for each column
cout << "| " << setw(nameWidth) << "Alice";
  1. Add spaces and | characters for visual borders

Fixed Width Table: Complete Code (Part 1)

//FixWwBorders.cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    const int nameWidth = 15;
    const int ageWidth = 10;
    const int salaryWidth = 12;
    const int totalWidth = nameWidth + ageWidth + salaryWidth + 8;
    const string title = "Fixed Width Table with Borders";
    
    cout << title << endl;
    cout << setfill('=') << setw(totalWidth) << "=" << endl;
    cout << setfill(' ')
         << "| " << left << setw(nameWidth) << "Name"
         << "| " << setw(ageWidth) << "Age"
         << "| " << setw(salaryWidth) << "Salary" << " |" << endl;
    cout << setfill('=') << setw(totalWidth) << "=" << endl;

    cout << setfill(' ')
         << "| " << setw(nameWidth) << "Alice"
         << "| " << setw(ageWidth) << 30
         << "| " << setw(salaryWidth) << 50000.50 << " |" << endl;
    
    cout << "| " << setw(nameWidth) << "Bob"
         << "| " << setw(ageWidth) << 25
         << "| " << setw(salaryWidth) << 45000.75 << " |" << endl;
    
    cout << "| " << setw(nameWidth) << "Charlie"
         << "| " << setw(ageWidth) << 28
         << "| " << setw(salaryWidth) << 52000.00 << " |" << endl;
    
    cout << setfill('=') << setw(totalWidth) << "=" << endl;
    cout << "End of Report." << endl;
    
    return 0;
}

Fixed Width Table: Important Concepts

1. Calculate total width:

const int totalWidth = nameWidth + ageWidth + salaryWidth + 8;
// +8 accounts for the "| | | |" characters and spaces

2. Create border lines with setfill:

cout << setfill('=') << setw(totalWidth) << "=" << endl;
// Fills the width with '=' characters

3. Always reset fill character:

cout << setfill(' ')  // Reset to space after using setfill('=')

4. Each column needs setw():

cout << "| " << setw(nameWidth) << "Alice"  // setw() only affects next item!
     << "| " << setw(ageWidth) << 30;        // Need setw() again

Table Type 2: Hexadecimal and Octal Table

When to Use

  • Show number system conversions (decimal, hex, octal)
  • Display memory addresses
  • Computer science / low-level programming data
  • Compare different representations of the same value

Example Output

+ Value Representations +
Format         Value
-----------------------
Decimal:       255
Hex:           FF
Octal:         377
-----------------------
+ End of Report +

Hex/Oct Table: Key Techniques

  1. Use dec, hex, and oct manipulators
cout << dec << value;  // Decimal
cout << hex << value;  // Hexadecimal
cout << oct << value;  // Octal
  1. Use uppercase for uppercase hex letters
cout << hex << uppercase << value;  // FF instead of ff
  1. Simple dashed borders

Hex/Oct Table: Complete Code

//HexOctTable.cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int value = 255;
    const int labelWidth = 12;
    const int valueWidth = 8;
    
    cout << "+ Value Representations +" << endl;
    cout << left << setw(labelWidth) << "Format"
         << left << setw(valueWidth) << "Value" << endl;
    cout << "-----------------------" << endl;
    
    cout << left << setw(labelWidth) << "Decimal:"
         << left << setw(valueWidth) << dec << value << endl;
    
    cout << left << setw(labelWidth) << "Hex:"
         << left << setw(valueWidth) << hex << uppercase << value << endl;
    
    cout << left << setw(labelWidth) << "Octal:"
         << left << setw(valueWidth) << oct << value << endl;
    cout << "-----------------------" << endl;
    cout << "+ End of Report +" << endl;
    
    return 0;
}

Number System Manipulators

Manipulator Purpose Example
dec Decimal (base 10) 255
hex Hexadecimal (base 16) ff
oct Octal (base 8) 377
uppercase Uppercase hex letters FF instead of ff

Remember: These manipulators are persistent - they stay active until changed!

cout << hex << 255 << endl;  // ff
cout << 100 << endl;         // 64 (still in hex mode!)
cout << dec << 100 << endl;  // 100 (back to decimal)

Cash Table: Key Techniques

  1. Use fixed and setprecision(2) for money
cout << fixed << setprecision(2);
  1. Handle negative numbers with conditional formatting
cout << "( $" << -groceries << ")";
  1. Use right alignment for numbers
cout << right << setw(15) << salary;
  1. Adjust setw() for parentheses
cout << setw(colWidth - 6) << "( $" << amount << ")";

Cash Table: Complete Code (Part 1)

//CashTablewNegs.cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    const int colWidth = 15;
    const int totalWidth = colWidth * 2 + 6;
    
    double salary = 1500.75;
    double groceries = -450.50;
    double freelance = 2000.00;
    double rent = -300.00;
    double investment = 750.25;
    
    cout << setfill('=') << setw(totalWidth) << "=" << endl;
    cout << setfill(' ')
         << "| " << left << setw(colWidth) << "Description"
         << "| " << setw(colWidth) << "Cash Amount" << " |" << endl;
    cout << setfill('-') << setw(totalWidth) << "-" << endl;
    
    cout << setfill(' ');
    cout << fixed << setprecision(2);

    cout << "| " << setw(colWidth) << left << "Salary";
    cout << "| " << setw(colWidth) << right << salary << " |" << endl;
    
    cout << "| " << setw(colWidth) << left << "Groceries";
    cout << "|" << setw(colWidth - 6) << right << "( $" << -groceries << ") |" << endl;
    
    cout << "| " << setw(colWidth) << left << "Freelance Work";
    cout << "| " << setw(colWidth) << right << freelance << " |" << endl;
    
    cout << "| " << setw(colWidth) << left << "Rent";
    cout << "|" << setw(colWidth - 6) << right << "( $" << -rent << ") |" << endl;
    
    cout << "| " << setw(colWidth) << left << "Investment";
    cout << "| " << setw(colWidth) << right << investment << " |" << endl;
    
    cout << setfill('=') << setw(totalWidth) << "=" << endl;
    
    return 0;
}

Cash Table: Negative Number Formatting

Regular output:

cout << -450.50;  // Displays: -450.5

Accounting format with parentheses:

// Store as negative, display with parentheses
double groceries = -450.50;
cout << "( $" << -groceries << ")";  // Displays: ( $450.50)

Why setw(colWidth - 6)? - colWidth is 15 - ( $ takes 3 characters - ) takes 1 character
- We need space for the number: 15 - 6 = 9 characters for the amount

Table Type 4: Scientific Notation Table

When to Use

  • Very large numbers (millions, billions, trillions)
  • Very small numbers (0.00001, etc.)
  • Scientific or engineering data
  • Precise notation regardless of scale

Example Output

Scientific Notation Output:
---------------------------------
Value 1:         1.235e+03
Value 2:         9.87654e+03

Scientific Table: Key Techniques

  1. Use scientific manipulator
cout << scientific << value;
  1. Use setprecision() to control significant figures
cout << scientific << setprecision(3) << value;
  1. Simpler table structure (fewer columns)

Scientific Table: Complete Code

// SciNotWLeft.cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    double num1 = 1234.56789;
    double num2 = 9876.54321;
    string title = "Scientific Notation Output:";
    
    cout << title << endl;
    cout << string(title.size(), '-') << endl;
    
    cout << left << setw(15) << "Value 1:"
         << scientific << setprecision(3) << num1 << endl;
    
    cout << setw(15) << "Value 2:"
         << scientific << setprecision(5) << num2 << endl;
    
    return 0;
}

Scientific Notation Explained

Scientific notation format: 1.234e+03 - Coefficient: 1.234 - Exponent: +03 (means × 10³) - Value: 1.234 × 1000 = 1234

Controlling precision:

double num = 1234.56789;

cout << scientific << setprecision(3) << num;  // 1.235e+03
cout << scientific << setprecision(5) << num;  // 1.23457e+03

Uppercase E:

cout << scientific << uppercase << num;  // 1.235E+03

Choosing the Right Table Type

Data Type Best Table Why
Mixed text and numbers Fixed Width with Borders Clear separation, professional
Number conversions Hex/Oct Table Shows different representations
Money/Budget data Cash Table Proper negative number formatting
Very large/small numbers Scientific Notation Consistent scale
Memory addresses Hex/Oct Table Hexadecimal format

Common Table Patterns: Column Widths

Always define column widths as constants at the top:

const int nameWidth = 15;
const int ageWidth = 10;
const int salaryWidth = 12;
const int totalWidth = nameWidth + ageWidth + salaryWidth + 8;

This makes it easy to adjust your table later!

Common Table Patterns: Border Lines

Use setfill and setw to create lines:

// Equals border
cout << setfill('=') << setw(totalWidth) << "=" << endl;

// Dash border
cout << setfill('-') << setw(totalWidth) << "-" << endl;

// Always reset
cout << setfill(' ');

Common Table Patterns: Headers and Data

Header row:

cout << setfill(' ')
     << "| " << left << setw(nameWidth) << "Name"
     << "| " << setw(ageWidth) << "Age"
     << "| " << setw(salaryWidth) << "Salary" << " |" << endl;

Data rows:

cout << "| " << setw(nameWidth) << "Alice"
     << "| " << setw(ageWidth) << 30
     << "| " << setw(salaryWidth) << 50000.50 << " |" << endl;

Building a Table: Step-by-Step Process

  1. Plan: What data? How many columns? Widest values?
  2. Define constants: Column widths and total width
  3. Set formatting: fixed, scientific, hex, etc.
  4. Create title and top border
  5. Add header row
  6. Add separator line
  7. Add data rows
  8. Add bottom border

Key Takeaways

  • The data determines the table type - Choose format based on your data
  • Plan before coding - Define column widths first
  • setw() resets - Use it for every column
  • setfill() persists - Always reset to space

Key Takaways

  • Output with cout. The cout statement displays data to the screen using the insertion operator <<, allows chaining multiple items together, and requires text to be enclosed in double quotes while variables display their values directly.

  • Input with cin. The cin statement reads user input using the extraction operator >>, waits for the user to press Enter before continuing, and stops reading at whitespace characters like spaces.
