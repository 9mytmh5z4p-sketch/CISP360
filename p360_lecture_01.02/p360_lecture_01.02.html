An Overview of Variable Types

Major C++ Variable Types

  • Integers
  • Floating Point
  • Char
  • Strings
  • Booleans
  • Void
  • Automatic Typing

Why Does C++ Have So Many Types?

  • Control
  • Multiple types allow the program designer (you) to precisely control how much memory will be reserved and what behaviors are allowed with the data.
  • This is not that much of a concern today.

Let’s briefly look at each type in detail.

Close-up on Integers

  • negative and positive numbers
  • just positive numbers
  • No decimals in either case.

Unsigned Integers

  • We can make an integer hold larger numbers by making it unsigned.
  • Downside - it can only hold positive numbers.
    • Many errors are made on either side of zero.
    • unsigned integers with a negative number will cause logic errors.
#include <iostream>
#include <string>

int main() {
    // Integer types
    int intNum = 42;
    unsigned int unsignedNum = 100;
    long long longLongNum = 9000000000LL;
    
    // Floating-point types
    float floatNum = 3.14f;         // good  to about 6 decimal places
    double doubleNum = 3.14159;     // good to about 15 decimal places
    
    // Character types
    char charVar = 'A';
    
    // Boolean type
    bool boolVar = true;
    
    // String type
    std::string stringVar = "Hello, World!";
    
    // Auto keyword (type inference)
    auto autoInt = 42;
    auto autoDouble = 3.14;
    auto autoString = std::string("Auto");
    
    // Const variable
    const int CONSTANT = 100;
    
    std::cout << "All major variable types demonstrated." << std::endl;
    
    return 0;
}

Close up on void

  • void means no type.
  • Incomplete type: A variable type which has been declared, but not defined.
  • The compiler knows about it but can’t do anything with it like declare memory.
  • Cannot be instantiated.

Not Allowed

void someVar;    // compiler error!
void stuff = ??;    // compiler error!

We use void for functions and pointers.

You’ll see it again.

Variable Types

Remember

C++ provides multiple data types (integers, floating-point numbers, characters, strings, and booleans) to give programmers precise control over memory usage and data behavior, though choosing the right integer or floating-point size is less critical today than it once was. The auto keyword allows the compiler to automatically infer variable types from initialization values, while const creates read-only variables that cannot be changed after they’re set.

Variables and Assignment

Why Do This?

  • Variables are used to store data.
  • Under program control (unlike literals).

Setting up a variable

  • Recall we need to tell the computer to set up a variable (declaration).
  • We do not need to assign data to it (initialization).
  • But initializing is a really, really good idea.

Declaration, Initialization, and Assignment

int x;      // Declaration, no initialization
int y = 25; // Declaration + Initialization

y = 30;     // Assignment, as variable already exists

= Does not mean what you think it does

In programming = is not the equals sign.

It’s called the Assignment Operator and it means we are moving data around in memory.

Remember

Variables are the key: If we don’t declare and initialize our variables, there’s very little we can do.

// Division
int quotient = 10 / 5;  // quotient = 2

Integer Division

You didn’t get this in elementary school!

Whenever you divide two integers together you get an integer!

Integer division drops the fractions.

int x = 7/2;      // 3, not 3.5
int y = 1/2;      // 0 (surprise!)

Division by Zero

  • Crash your program (if you are lucky).
  • Logic errors (if you are not).

Mixed Operations

We will discuss what happens when you perform a mathematical operation between different data types later.

Operator Precedence

Arithmetic operators follow the same order of operations you learned in math class.

Order (highest to lowest priority):

  1. Parentheses ()
  2. Multiplication *, Division /, Modulus %
  3. Addition +, Subtraction -

Examples:

int result1 = 5 + 3 * 2;      // result1 = 11 (not 16)
int result2 = (5 + 3) * 2;    // result2 = 16
int result3 = 10 - 6 / 2;     // result3 = 7 (not 2)

Tip: When in doubt, use parentheses to make your intentions clear!

Compound Assignment Operators

Shortcuts for modifying a variable using arithmetic operations.

Syntax: variable operator= value is equivalent to variable = variable operator value

Operator Meaning Example Equivalent To
+= Add and assign x += 5 x = x + 5
-= Subtract and assign x -= 3 x = x - 3
*= Multiply and assign x *= 2 x = x * 2
/= Divide and assign x /= 4 x = x / 4
%= Modulus and assign x %= 3 x = x % 3

Example:

int score = 100;
score += 50;   // score is now 150
score -= 20;   // score is now 130
score *= 2;    // score is now 260

The cmath Library

C++ provides a powerful math library with advanced mathematical functions beyond basic arithmetic.

To use it, include at the top of your program:

#include <cmath>

Common functions:

Function Purpose Example Result
sqrt(x) Square root sqrt(16) 4.0
pow(x, y) x raised to power y pow(2, 3) 8.0
abs(x) Absolute value abs(-5) 5
ceil(x) Round up ceil(4.3) 5.0
floor(x) Round down floor(4.7) 4.0
round(x) Round to nearest round(4.5) 5.0

Example:

double radius = 5.0;
double area = M_PI * pow(radius, 2);  // area = 78.54

Remember

C++ arithmetic operators work like basic math but with important programming-specific behaviors: integer division truncates decimals, the modulus operator (%) returns remainders, and compound assignment operators (like +=) provide convenient shortcuts for updating variables. For advanced mathematical functions beyond the five basic operators, the cmath library offers tools like square roots, powers, and rounding functions.

Key Takaways

  1. Data Types & Memory Control: C++ uses specific types (like int, double, and bool) to allow programmers to precisely manage memory reservation and data behaviors.

  2. Variable Management: Variables must be declared before use, while keywords like const (for read-only data) and auto (for type inference) streamline assignment logic.

  3. Arithmetic Logic: Mathematical operations follow standard algebraic precedence, with integer division resulting in truncation and the modulus operator (%) used to find remainders.
